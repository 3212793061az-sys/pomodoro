🍅 Pomodoro Pro

Aplicación web de productividad basada en la técnica Pomodoro para gestionar tareas y mejorar la concentración.

📌 Características
⏱️ Temporizador Pomodoro (25 min trabajo / 5 min descanso)
🔁 Ciclos automáticos con descanso largo configurable (15 o 30 min)
📝 Gestión de tareas (bloqueada durante ejecución)
⏯️ Controles: iniciar, pausar, detener, siguiente
💾 Persistencia con localStorage
📜 Historial de tareas con fecha y duración
🔔 Notificaciones al cambiar de bloque
🧠 Mantiene estado incluso si cambias de pestaña
🧠 ¿Cómo funciona?
Escribes una tarea
Inicias el temporizador
Trabajas durante 25 minutos
Descansas 5 minutos
Cada 4 ciclos → descanso largo
Al detener, se guarda el historial automáticamente
